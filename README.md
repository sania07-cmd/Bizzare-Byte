**Bizzare-Byte** is a full-stack **College Event Management System** designed to simplify the process of managing college-level events. The platform allows administrators and participants to efficiently handle event creation, registrations, and basic event-related workflows through a responsive web interface.

The project follows a **separation of concerns** architecture, with a clean frontend, a REST-based backend, and a relational database for persistent storage.
